NO INSTALL NEEDED
Run with npx

Have Node.js? Run the CLI instantly without installing. No PATH changes required.

$ npx fluently-cli score "your task description"
$ npx fluently-cli compare --description "AI code review" --delegation augmented
💾
PERMANENT INSTALL
Install Globally

Add fluent to your PATH once and use it everywhere in your terminal workflow.

$ npm install -g fluently-cli
$ fluent score "review my AI prompt strategy"

Requires Node.js 20+. The CLI is self-contained — no configuration files or environment variables needed.

How It Works

Three steps to turn any AI task into structured, accountable practice.

1

Identify your AI task

Write a clear description of what you want AI to do and your intended level of delegation — full automation, augmentation, or supervision only.

2

Score with Fluently

The in-browser scoring engine uses keyword-cosine similarity to match your task against 16 curated cycles across 7 domains, ranking by dimensional fit.

3

Apply the matching cycle

Review the top matching cycles, inspect each dimension's guidance, and adapt the patterns to your specific context. Contribute your refinements back to the community.
